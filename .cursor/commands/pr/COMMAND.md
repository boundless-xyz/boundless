Create a pull request for the current changes.
1. Look at the staged and unstaged changes with `git diff`
2. Write a clear commit message based on what changed (use commit message style)
3. Commit and push to the current branch
4. Use `gh pr create --web --title <title> --body <body>` to generate a pull request URL with title/description (use description template below)
5. Return the PR URL when done

# Commit message style
* Simple, one sentence

Changes
